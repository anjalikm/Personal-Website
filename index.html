15 Square Puzzle

A classic game using Priority Queue

This is developed in JavaFX. It has 2 aspects. First, the user can play the game while sliding the tiles and the game tells if he gets the ansers. Second, the user can click on 'Solution' and the games shows the steps animated to the the solution. The game uses Priority queue where each node contains the upto 4 possible next moves the manhatten distance for each possible move. This distance is used to pick the next move with least value.

  • Date: July 2014
  • Fields: Algorithms, JavaFX, Graphics
  • Code: Gitgub Repo.
